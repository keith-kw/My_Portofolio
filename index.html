    RAAMEN

    RAAMEN is a project from Pattern Design Software course, where RAAMEN is a web-based online ramen selling application. The aim of this project is able to create a website is by using ASP.NET and Domain Driven Design (DDD) method for developing the website and also using SAP Crystal Report for the Transaction Report. This website can be accessed by 3 roles, which are Member that can buy and view the menu and access the purchase history. Staff can handle the Member order, and insert, update and delete the ramen, and also view all the customer list. Also, Admin have both Member and Staff privilege, and view all the transaction reports. On the transaction history when the user click the detail button it will redirect to the transaction detail page using URL Query String. The challenges while working on this project are making sure the the view on the Crystal report is correct and also applying the DDD method correctly, which I as the project developer managed to overcome it by studying and finding more references from similar and online discussion.

    Si Paling Cerdas

    Si Paling Cerdas is a desktop application and an educational game based on a quiz game where players can compete with each other on various knowledge topics. Every match have a certain duration which is 5 minutes and if the players answer the question correctly, a point will be added to their score. From all the accumulated score, player can get into the leaderboard or see other players on the leaderboard .This game is made for Software Engineering Course. The challenges as the designer of the project are to create a good UI/UX design and ensure that all the features required by the user have been fulfilled. The challenges as the project manager are ensuring smooth communication among team members, planning the project such as defining the scope, making the schedule step based on the Gantt chart, and evaluating whether the final result has fulfilled all the requirements which had been agreed beforehand.
